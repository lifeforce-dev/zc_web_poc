2. Block Design

Finalized

Blocks are the primary game pieces. Each block displays four power values (N, E, S, W) and must clearly communicate both the owner's elemental type AND the player who owns it.

Design Decisions:
  • Elemental Identity: Center icon in elemental color with glow effect (50% opacity)
  • Player Ownership: 2px border in player color (blue/orange)
  • Numbers: Gold tablets floating outside block edges
  • Corner Accents: Gold L-shaped brackets in corners
  • Base Style: Dark stone tablet gradient

3. Game Board Design

Finalized

The game board uses the "Void Storm v2" wall design - animated WebGL2 walls with swirling purple clouds and diffuse lightning flashes. Playable tiles use a dark stone gradient.

Design Decisions:
  • Board Background: Near-black (#050404) with subtle border
  • Tile Gap: 12px between tiles
  • Playable Tiles: Dark stone gradient with subtle border
  • Wall Tiles: WebGL2 animated void storm effect
  • Tile Hover: Lift + gold border glow
